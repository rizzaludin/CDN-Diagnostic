v1.0

Network diagnostic tool untuk NOC Engineer. Monitor CDN traffic, analisis DNS propagation, ukur latency, dan pantau routing BGP secara real-time.

Fitur

  • Real-time IP monitoring & WHOIS lookup
  • ASN/CDN brand identification (20+ built-in)
  • BGP Streaming real-time via RIS Live (RIPE NCC)
  • DNS Propagation Checker (6 resolvers)
  • Latency & Continuous Ping Monitor
  • BGP Route Information (RIPE Stat API)
  • Traceroute visualization
